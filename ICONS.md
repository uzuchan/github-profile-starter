# Skill icons checklist / 使用技術アイコン チェックリスト

Check the icons you actually use by changing `- [ ]` to `- [x]`, then run
`bash scripts/fetch-skills.sh`. The checked icons are turned into the skill
strips shown in your profile (one strip per category below).

使ったことのあるものを `- [ ]` → `- [x]` に変えて `bash scripts/fetch-skills.sh` を
実行してください。チェックした分だけがプロフィールのスキル帯に反映されます（下のカテゴリごとに1帯）。

- Names are skillicons.dev IDs (all lowercase). Full list: <https://skillicons.dev>
- Not listed? Just add a line like `- [x] **<id>**` under the right category.
- アイコン名は skillicons.dev のID（すべて小文字）。一覧は上記。無いものは行を足せばOK。
